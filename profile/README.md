## Social Dept.

We're a product design & development studio focused on building products and tools for the social web. Developing high-quality open-source packages and services designed to support emerging web standards, specializing in AT Protocol.

---

### Product Lineup

Below is a list of products we've built:

| Package                                      | Description                                                                                |
|----------------------------------------------|--------------------------------------------------------------------------------------------|
| [**Offprint**](https://offprint.app)         | Publishing platform for longform content deeply integrated with the AT Protocol ecosystem. |
| [**Social Headers**](https://headers.social) | Growing & curated directory of the best designed profile banners across the ATmosphere.    |

We're continually building new products that strengthen the social web and improve the experience for creators, developers, and organizations.

---

### Open Source Packages

Below is a list of core repositories we've published:

| Package                                                            | Description                                                                                                                      |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**atp-resolver**](https://github.com/socialdept/atp-resolver)     | Resolve AT Protocol DIDs, handles, and schemas with intelligent caching for Laravel.                                             |
| [**atp-schema**](https://github.com/socialdept/atp-schema)         | Parse and validate AT Protocol Lexicons with DTO generation for Laravel.                                                         |
| [**atp-signals**](https://github.com/socialdept/atp-signals)       | Build Reactive Signals for Bluesky's AT Protocol Firehose in Laravel.                                                            |
| [**color-scales**](https://github.com/socialdept/color-scales)     | Generate Tailwind color palettes from any color in Laravel.                                                                      |
| [**forward.direct**](https://github.com/socialdept/forward.direct) | Open source request forwarding webservice built on Cloudflare Workers. Inspired by [fwd.host](https://fwd.host) by Laravel Herd. |

We're always developing more Laravel packages as we find lack-luster or no solutions available in the ecosystem.
